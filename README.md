# Menu de login básico com validador de senha

Criação de um pequeno sistema de cadastro e login com validador de senha no terminal do Visual Studio Code com Python 3

## Funcionalidades

- Cadastro de usuário
- Validação de senha
  - mínimo de 8 caracteres
  - pelo menos uma letra
  - pelo menos um número
- Sistema de login
- Limite de tentativas
- Bloqueio após tentativas excedidas

## Tecnologias

- Python 3

## Como executar

1. Certifique-se de ter o Python instalado
2. No terminal, dentro da pasta do projeto, execute:

```bash
python main.py
```
